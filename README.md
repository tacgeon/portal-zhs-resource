# portal-zhs-resource

## 项目简介

`portal-zhs-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或具体网站，仅作为静态页面资源的集合与分发点。

## 目录结构

```
portal-zhs-resource/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立的页面归档单元，内部包含对应的 HTML 文件及相关资源。
- `assets/` 目录用于存放多个页面可能共同使用的公共静态资源，以减少重复。

## 页面归档说明

- 每个独立 HTML 页面均以文件夹形式归档于 `pages/` 下，文件夹名称即为页面标识。
- 页面内所有资源引用均使用相对路径，确保在本地或静态托管环境中均可正常访问。
- 归档页面可能包含说明性注释或版本标记，以便追溯更新历史。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会在 `pages/` 下添加或更新对应文件夹。
- 提交说明应简要描述变更内容，例如“新增 XX 页面归档”或“更新 XX 页面资源”。
- 不欢迎包含任何营销、诱导点击或违反相关法律法规的内容。
- 如发现页面存在问题（如链接失效、资源缺失），欢迎提交 Issue 反馈。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/portal-zhs-resource.git
   ```
2. 直接通过浏览器打开 `pages/` 下任意文件夹内的 HTML 文件即可查看对应页面。
3. 也可将本仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等），通过 URL 访问对应页面路径。
